## 🎯 系列目标

通过这个系列项目，你将学会：

1. **Flutter 基础** - Widget、布局、导航
2. **状态管理** - Provider 的使用
3. **网络请求** - Dio 的封装和使用
4. **数据持久化** - SharedPreferences 的使用
5. **MVVM 架构** - 分层架构的设计
6. **原生集成** - MethodChannel 的使用
7. **UI 设计** - Material 3 的应用
8. **最佳实践** - 代码规范、错误处理
